# LegalClaw

Legal practice management for [ERPClaw](https://github.com/avansaber/erpclaw). Matter management, time and billing, IOLTA trust accounting, document management, court calendar, conflict checking, and bar/CLE compliance. 69 actions across 7 domains.

## Features

- **Matters** -- client intake, matter lifecycle, party tracking, matter summaries, client portfolios
- **Time & Billing** -- time entries (6-minute increments), expenses, invoice generation, payment recording, AR aging
- **Trust Accounting** -- IOLTA/escrow accounts, deposits, disbursements, transfers, reconciliation, interest distribution
- **Documents** -- legal document management, versioning, court filing, archival, full-text search
- **Calendar & Deadlines** -- court dates, statute of limitations, reminders, critical deadline tracking
- **Conflict Checking** -- name-based conflict search, waiver management, conflict reports
- **Compliance** -- bar admissions, CLE tracking, compliance reports, practice area analysis, profitability reports

Trust accounts enforce balance sufficiency checks on every disbursement. Invoice generation creates real sales invoices via ERPClaw cross-skill integration.

## Installation

Requires [ERPClaw](https://github.com/avansaber/erpclaw) core.

```
install-module legalclaw
```

## Quick Start

```
"I'm starting a law firm -- set me up"
```

The bot will install LegalClaw and guide you through client intake, matter creation, time tracking, and trust account setup.
